# Felhő Alapú Raktárkezelő Rendszer

Ez egy modern, reszponzív raktárkezelő alkalmazás, amelyet kisvállalkozások számára fejlesztettem. 
A rendszer alkalmas készletek nyomon követésére, kategóriák kezelésére és automatikus készletfigyelmeztetések küldésére.

## Főbb funkciók
- **Készletnyilvántartás:** Termékek hozzáadása, szerkesztése és törlése.
- **Kategóriák:** Termékek csoportosítása a könnyebb átláthatóságért.
- **Készletfigyelő:** Automatikus figyelmeztetés (⚠️), ha egy termék eléri a kritikus szintet.
- **Export:** Havi jelentések letöltése CSV formátumban.
- **Reszponzív design:** Mobilról és asztali gépről is tökéletesen használható.

## Technikai háttér
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid elrendezés)
- **Adatbázis:** SQLite
- **Hitelesítés:** Flask-Login (Admin és Felhasználói szintek)
